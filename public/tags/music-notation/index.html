I chuckled to myself, because this seems kind of strange to show up into sample names. Why is this strange you may ask? Well, for starters the auto sampler uses both the note names, as well as the midi note numbers in the file name when you auto sample an instrument. This makes sense, even if it is duplicate information. However, when you do multiple velocity levels, the auto sampler adds f/fm/pm/p (or a subset there of) into the file names. Strange huh?
